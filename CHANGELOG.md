# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-15

### Added
- Fullstack Team To-do app with RBAC (`ADMIN`, `MEMBER`).
- NestJS backend with JWT authentication and Swagger docs.
- Prisma + MySQL data model for users and tasks.
- Task CRUD APIs with role-based authorization rules.
- React + Vite frontend dashboard with login/register and task board.
- Demo seed script at `prisma/seed.ts`.
- Docker files (`Dockerfile`, `frontend/Dockerfile`, `docker-compose.yml`).
- Backend unit tests for auth, tasks service, and roles guard.
- Frontend Vitest tests for auth page, task board, and storage helpers.
- GitHub Actions CI workflow for lint/test/build on backend and frontend.
